# Security Policy

## Supported Versions

We currently support the following versions of Laika Music with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of Laika Music seriously. If you believe you have found a security vulnerability, please report it to us by opening a private security advisory on GitHub or by contacting the maintainers directly.

**Please do not report security vulnerabilities through public GitHub issues.**

### Our Process

1.  **Submission**: You submit a report.
2.  **Acknowledgment**: We will acknowledge your report within 48 hours.
3.  **Investigation**: We will investigate the issue and determine its impact.
4.  **Resolution**: We will work on a fix and release it as soon as possible.
5.  **Disclosure**: We will coordinate the public disclosure of the vulnerability.

Thank you for helping keep Laika Music secure!
