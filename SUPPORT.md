# Support

We want to ensure you have a great experience with Laika Music. If you need help, here are the best ways to get support.

## Documentation

Before seeking support, please check the following resources:

- **README**: The [README.md](README.md) file contains basic setup and usage instructions.
- **Issues**: Search existing [GitHub Issues](https://github.com/your-username/laika-music/issues) to see if your question has already been answered.

## Community Support

If you have questions or need help with something not covered in the documentation:

- **GitHub Discussions**: Use [GitHub Discussions](https://github.com/your-username/laika-music/discussions) for general questions, ideas, and community support.

## Reporting Bugs

If you've found a bug, please [open an issue](https://github.com/your-username/laika-music/issues/new/choose) using the appropriate template.

## Professional Support

If you require professional support or have business inquiries, please contact the maintainers directly.
